# 再コール時間の再設定

この記事では、再コール時間の再設定方法を2つご説明します。

**目次**\
[**再コールのポップアップからの再設定**](13593294000153_再コール時間の再設定.md#h_01GMVV5NN4VQ8FH5VXEWSDJVRG)\
[**再コールリストからの再設定**](13593294000153_再コール時間の再設定.md#h_01GMVV5W8JYGBCKGYM0XJBKBJB)

## **再コールのポップアップからの再設定**

1. 再コールを設定している日時になると「再コール詳細内容」のポップアップが表示されます。\
   ![スクリーンショット 2024-02-13 23.45.39.png](../../.gitbook/assets/13593294000153_0.png)
2. 再コールの日時を改める場合は、ポップアップ内の一番下「時刻再設定」の左側の枠にカーソルをあわせます。\
   カーソルをあわせると、カレンダー（赤枠）が表示されます。\
   カレンダー上で日付・時刻（15分刻み）の設定が可能です。\
   ![スクリーンショット 2024-02-13 23.46.00.png](../../.gitbook/assets/13593294000153_1.png)
3. 15分刻みではなく細かく指定されたい場合は、赤枠内へ直接入力変更が可能です![スクリーンショット 2024-02-13 23.48.46.png](../../.gitbook/assets/13593294000153_2.png)
4. 赤枠内の「5分後に再通知」をクリックすると、5分後に再度ポップアップが表示されます。![スクリーンショット 2024-02-13 23.48.46 (1).png](../../.gitbook/assets/13593294000153_3.png)
5. 「時刻再設定」もしくは「5分後に再通知」の再設定が完了すると「時刻を再設定しました。」とポップアップが表示されます。\
   ![スクリーンショット 2024-02-14 10.53.20.png](../../.gitbook/assets/13593294000153_4.png)

## **再コールリストからの再設定**

1. 再コールリストを開きます。\
   ![スクリーンショット 2024-02-14 11.09.21.png](../../.gitbook/assets/13593294000153_5.png)
2. 再コールリストの中で日時の再設定を行いたいリストの鉛筆マークをクリックします。\
   ![スクリーンショット 2024-02-14 11.05.41.png](../../.gitbook/assets/13593294000153_6.png)
3. 再コールの日時を改める場合は、ポップアップ内の一番下「時刻再設定」の左側の枠にカーソルをあわせます。\
   カーソルをあわせると、カレンダーが表示されます。\
   カレンダー上で日付・時刻（15分刻み）の設定が可能です。![スクリーンショット 2024-02-14 11.27.43.png](../../.gitbook/assets/13593294000153_7.png)
4. 15分刻みではなく細かく指定されたい場合は、赤枠内へ直接入力変更が可能です。\
   ![スクリーンショット 2024-02-14 11.06.22.png](../../.gitbook/assets/13593294000153_8.png)
5. 赤枠内の「5分後に再通知」をクリックすると、5分後に再度ポップアップが表示されます。![スクリーンショット 2024-02-14 11.06.02.png](../../.gitbook/assets/13593294000153_9.png)
6. 再設定が完了すると「時刻を再設定しました。」とポップアップが表示されます。\
   ![スクリーンショット 2024-02-14 11.38.58.png](../../.gitbook/assets/13593294000153_10.png)

### 編集ボタンから再設定

1. 再コールリストを開き、再コールを行いたいリストにチェックをし赤枠内「編集」ボタンをクリックします。\
   ![スクリーンショット 2024-02-14 11.06.40.png](../../.gitbook/assets/13593294000153_11.png)
2. 再コール詳細内容が表示され、ポップアップ内の一番下「時刻再設定」の左側の枠にカーソルをあわせます。\
   カーソルをあわせると、カレンダーが表示されます。\
   カレンダー上で日付・時刻（15分刻み）の設定が可能です。\
   ![スクリーンショット 2024-02-14 13.18.47.png](../../.gitbook/assets/13593294000153_12.png)
3. 5分刻みではなく細かく指定されたい場合は、赤枠内へ直接入力変更が可能です。\
   ![スクリーンショット 2024-02-14 13.23.43.png](../../.gitbook/assets/13593294000153_13.png)
4. 赤枠内の「5分後に再通知」をクリックすると、5分後に再度ポップアップが表示されます。\
   ![スクリーンショット 2024-02-14 13.27.00.png](../../.gitbook/assets/13593294000153_14.png)
5. 再設定が完了すると「時刻を再設定しました。」とポップアップが表示されます。\
   ![スクリーンショット 2024-02-14 13.29.22.png](../../.gitbook/assets/13593294000153_15.png)
