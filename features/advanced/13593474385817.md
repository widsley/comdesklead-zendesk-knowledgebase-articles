# コールモード上に設置してある2つの保存ボタンの違い

コール画面にある保存ボタンは2つあります。（赤枠）

・リスト項目の保存

・アクティビティ結果の保存

![\_\_\_\_\_\_\_\_\_\_2022-12-18\_14.29.40.png](../../.gitbook/assets/13593474385817_0.png)

## **リスト項目の保存**

リスト項目上にある保存ボタンは、赤枠内のいわゆる「リスト項目」に登録されている情報を編集し、保存する際に使用するボタンです。

![\_\_\_\_\_\_\_\_\_\_2022-12-18\_14.33.55.png](../../.gitbook/assets/13593474385817_1.png)

## **アクティビティ結果の保存**

アクティビティ結果（応対者・ステータス）の上にある保存ボタンは、架電終了後に赤枠内の架電結果を保存する際に使用するボタンです。

![\_\_\_\_\_\_\_\_\_\_2022-12-18\_14.37.15.png](../../.gitbook/assets/13593474385817_2.png)

編集した項目に従って、適切な保存ボタンをご利用ください。
