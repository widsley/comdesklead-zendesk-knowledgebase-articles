# IP回線で音声が聞こえない・聞こえづらい

IPで音声が聞こえない  
【ネットワークの記事】

【音声聞こえない・マイクが入らない】  
システム>サウンド>音量ミキサー>アプリ

chromeをクリックで基本既定  
（windows）設定>システム>サウンド>音量ミキサー>アプリ>出力デバイス・入力デバイスを対象のヘッドセットを選択

既定の場合でも、対象のものに接続されているとは限らない➡対象のヘッドセットを選択していただくようお願いいたします。  
  

【IP回線で音声が聞こえづらい】  
・パソコンが推奨スペックになっているか

┗コア i5以上・メモリ 8GB以上が望ましい　

※Comdesk Leadのみを起動させる場合の最低限のメモリです  
OS Windows10～11  
MAC

・パソコン側に負荷が掛かってないか。  
　➡負荷がかかっているとパソコン側で処理しきれなくなるため、音声品質に影響が出る可能性がございます。ブラウザを閉じたりし、負荷を減らした状態でご利用ください。

【イヤホンのタイプ】

・Bluetoothイヤホン  
┗電波が混線してしまうので、音声に影響しやすい。  
　在宅等で一人の環境でしたら、影響を受けづらくはあります。
