# ヘルプセンターログイン時のパスワードを忘れてしまった

ヘルプセンターログイン時のパスワードを忘れてしまった際の再設定方法についてご説明します。

1. 赤枠「パスワードを忘れた場合」をクリックします。\
   ![\_\_\_\_\_\_\_\_\_\_2022-11-28\_17.05.50.png](../../.gitbook/assets/12941407647001_0.png)
2. パスワードを再設定したいメールアドレスを入力し送信します。\
   ![\_\_\_\_\_\_\_\_\_\_2022-11-25\_12.42.42.png](../../.gitbook/assets/12941407647001_1.png)
3. 登録しているメールアドレス宛にパスワード再設定のメールが届きます。\
   メールのURLより、パスワードの再設定を行ってください。\
   ![\_\_\_\_\_\_\_\_\_\_2022-11-25\_12.43.13.png](../../.gitbook/assets/12941407647001_2.png)
4. 再設定後、メールアドレスと再設定したパスワードでログインを行ってください。

その他ご不明点などございましたら、[**サポートチームまでお問い合わせ**](https://comdesklead.zendesk.com/hc/ja/requests/new)をお願い致します。
