# Comdesk Leadにログインする

1.  Comdesk Leadにログインします。\
    ログインする際は必ずURL : [login.comdesk.com](http://login.comdesk.com) からログインします。

    また、ブラウザはGoogle Chromeをご利用ください。

    ![CDL\_login.png](../../.gitbook/assets/12735918031513_0.png)
2.  ユーザーIDとパスワードを入力し、「Log In」ボタンを選択します。

    ![CDL\_Login2.png](../../.gitbook/assets/12735918031513_1.png)

    ※ユーザーIDとパスワードは、ユーザーマッピング図で確認できます。

    （ユーザーマッピング図は御社管理者様にご提供しております。）

    ユーザーID・パスワードが不明な場合や、その他ご不明点などございましたら、[**サポートチームまでお問い合わせ**](https://comdesklead.zendesk.com/hc/ja/requests/new)をお願い致します。
