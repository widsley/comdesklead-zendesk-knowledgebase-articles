## **Myboxの解除について**

Myboxの解除は、Mybox担当者またはシステム管理者の権限を持つ全てのアカウントでこの作業が可能です。

解除方法は以下のとおりですので、それぞれご説明いたします。

・コール画面からの解除

　　∟Mybox担当者が自身のMyboxを解除する

　　∟管理者が他のユーザーのMyboxを解除する

・Mybox管理画面からMybox登録状態を解除する

## **コール画面からのMybox解除方法（自身のMyboxを解除する場合）**

1. コール画面でMyboxを表示し、解除したいリストを選択します。\
   ![\_\_\_\_\_\_\_\_\_\_2022-12-16\_16.16.15.png](../../.gitbook/assets/13556859672217_0.png)
2. 赤枠内の「Mybox」ボタンをクリックします。\
   ![\_\_\_\_\_\_\_\_\_\_2022-12-16\_16.16.28.png](../../.gitbook/assets/13556859672217_1.png)
3. 「マイボックスから外してもいいですか？」とポップアップが表示されます。\
   解除で間違いなければ「OK」をクリックします。\
   ![\_\_\_\_\_\_\_\_\_\_2022-12-16\_16.16.36.png](../../.gitbook/assets/13556859672217_2.png)
4. Myboxから解除されました。\
   ![\_\_\_\_\_\_\_\_\_\_2022-12-16\_16.16.55.png](../../.gitbook/assets/13556859672217_3.png)

## **コール画面からのMybox解除方法（管理者が他のユーザーのMyboxを解除する場合）**

1. 「マスターデータ管理」を開きます。\
   ![\_\_\_\_\_\_\_\_\_\_2022-12-21\_16.46.52.png](../../.gitbook/assets/13556859672217_4.png)
2. 他のユーザーのMyboxに入っている、解除したいリストを選択します。\
   ![\_\_\_\_\_\_\_\_\_\_2022-12-21\_16.47.03.png](../../.gitbook/assets/13556859672217_5.png)
3. 赤枠内の「Mybox」ボタンをクリックします。\
   ![\_\_\_\_\_\_\_\_\_\_2022-12-21\_16.47.17.png](../../.gitbook/assets/13556859672217_6.png)
4. 「マイボックスから外してもいいですか？」とポップアップが表示されます。\
   解除で間違いなければ「OK」をクリックします。\
   ![\_\_\_\_\_\_\_\_\_\_2022-12-21\_16.47.25.png](../../.gitbook/assets/13556859672217_7.png)
5. Myboxから解除されました。

## **Mybox管理画面からの解除方法**

1. Mybox管理画面に移動します。\
   ![\_\_\_\_\_\_\_\_\_\_2022-12-16\_16.22.06.png](../../.gitbook/assets/13556859672217_8.png)
2. Myboxから解除をしたいリストの左側のチェックボックスに✔を入れます。\
   ![\_\_\_\_\_\_\_\_\_\_2022-12-27\_10.34.26.png](../../.gitbook/assets/13556859672217_9.png)
3. 赤枠内「Mybox解除」ボタンをクリックします。\
   ![\_\_\_\_\_\_\_\_\_\_2022-12-27\_10.35.32.png](../../.gitbook/assets/13556859672217_10.png)
4. 「解除しました」とポップアップが表示されたら解除完了です。![\_\_\_\_\_\_\_\_\_\_2022-12-27\_10.36.36.png](../../.gitbook/assets/13556859672217_11.png)
