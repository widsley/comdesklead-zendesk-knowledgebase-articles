# 画面が正常に表示されない（レイアウトが崩れている）

特定の端末でのみ画面のレイアウトが正しく表示されずレイアウトが崩れている場合には、ブラウザのキャッシュのクリアをお試しください。

※キャッシュとは  
ブラウザが一度読み込んで表示したWebページの内容・情報を一時的に端末に保存しておく機能です。  
次に同じWebページにアクセスしたときに、その保存されたデータを参照することで素早く表示することができる一方、  
キャッシュに保存されたデータが古い場合、画面表示が崩れてしまう場合があります。

※Comdesk Leadでは下記ブラウザを推奨環境とさせていただいております。  
・[Google Chrome （最新版）](https://www.google.co.jp/chrome/?brand=FDKM&gclid=CjwKCAjw2bmLBhBREiwAZ6ugo6yBpAz825VcOfxptSQCEd8oH9F-wpRixujy4ldNbp_RI_Hz15hmRRoCBxgQAvD_BwE&gclsrc=aw.ds)

## **パソコンでのブラウザのキャッシュクリア方法**

### **ブラウザ：Google Chrome**

1.  画面右上の黒丸（もしくは横線）が縦に3つ並んでいるメニューボタンをクリックし、メニュー内にある「設定」を選択
2.  「プライバシーとセキュリティ」の項目にある「閲覧履歴データの削除」をクリック
3.  「キャッシュされた画像とファイル」のみにチェックを入れ、「データを消去」をクリック
4.  該当のページを再度読み込む

## **スマートフォンでのブラウザのキャッシュクリア方法**

### **OS：Android（ブラウザ：Chrome）**

1.  「Chrome」アプリを開いてメニューを選択し、「設定」をタップ
2.  「プライバシーとセキュリティ」をタップし、「閲覧履歴データの削除」をタップ
3.  「キャッシュされた画像とファイル」にチェックが入っていることを確認し、「データを削除」をタップ
4.  該当のページを再度読み込む

### **OS：iPhone（ブラウザ：Safari）**

1.  ホーム画面から「設定」をタップ
2.  「Safari」を選択
3.  「履歴と Web サイトデータを消去」をタップ
4.  「履歴とデータを消去」をタップ
5.  該当のページを再度読み込む
