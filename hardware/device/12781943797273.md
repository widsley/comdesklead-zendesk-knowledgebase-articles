# IMEIの確認方法

## **IMEIとは**

「IMEI」とは端末識別番号で、「国際移動体装置識別番号（International Mobile Equipment Identifier）」の頭文字です。

## **IMEIの確認方法**

1. スマートフォンの設定アイコンを選択します。\
   ![IMEI1.png](../../.gitbook/assets/12781943797273_0.png)
2. デバイス情報を選択します。\
   ![IMEI2.png](../../.gitbook/assets/12781943797273_1.png)
3. IMEIを選択します。\
   ![IMEI3.png](../../.gitbook/assets/12781943797273_2.png)
4. IMEIを確認します。\
   ![IMEI4.png](../../.gitbook/assets/12781943797273_3.png)
