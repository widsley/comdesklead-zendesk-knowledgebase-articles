# 2025/07/02　Comdesk Lead夜間リリースのお知らせ

平素より大変お世話になっております。Widsley Supportでございます。

いつもご利用ありがとうございます。

本日（2025/07/02）夜間リリースにて、Comdesk Leadに下記リリースを実施予定でございます。

挙動や仕様において、一部変更となる部分がございますので、ご認識いただけますと幸いです。

——————————————————————————–————————————————–————————–

## **■機能追加**\*\*【Web】\*\*

### 通話テキストの個人情報マスキング機能がテナント単位でON/OFFの設定が可能となります。

マスキング機能をOFFにご変更されたい場合は、テナント設定＞テナント設定編集　にある\
「通話テキストの個人情報マスク」のトグルからご変更をお願いいたします。

個人情報にあてはまるワード：（氏名・住所・組織・数詞）

※リリース後は、自動的に現状と同様の設定（ON）の状態となります。\
通話テキストへの反映は、テナント設定編集で設定変更後の通話からとなります。

——————————————————————————–————————————————–————————–

### ■不具合改修

【ComDesk Phone（デスクトップアプリ）】

* Web画面で設定したステータス・応対者の並び順がデスクトップでも反映されるようになりました。\
  ワークグループ設定で変更した並び順が適用されます。

【Web】

* ステータスの子項目を新規作成時、コール画面で子項目が重複して表示されてしまう不具合を修正いたしました。
* マスターデータ管理にて「システム項目なし」でエクスポートした際のファイル名が\
  「インポート」になっていたため、「エクスポート」へ変更いたしました。
