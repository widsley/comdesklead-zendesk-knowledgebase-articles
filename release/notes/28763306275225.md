# 2024/02/21アップデート予定　再コール機能について

2月21日夜間のリリースにて、再コールの機能がアップデート予定となっております。

大きく変更となる部分は以下の4つの部分となります。

### ◇再コール時、リストへ遷移せずダイアログ内での再コール

従来：再コールリスト・再コールのポップアップから再架電する際、リストへ遷移してからの発信。

アップデート後：リストへ遷移せず、再コール専用のダイアログから発信・履歴を残すことが可能に。

### ◇再コールリストを開いた際のリスト表示

従来：対応状況が「予約中/経過未対応/コール済み/キャンセル」全ての再コールが表示される。

アップデート後：対応状況「予約中/経過未対応/コール済み」のみ表示され「キャンセル」は除いて表示される。

### ◇再コールリスト内での絞り込み

従来：「ワークグループ/名前/予約状況/プロジェクト/登録者/指定日時/コール日時」での条件検索のみ。

アップデート後：従来の検索条件に加え、「プロジェクト/ステータス（複数選択可能）」での条件検索が可能に。

加えて、条件検索後のリストの中から検索が可能に。

### ◇時計マークの通知

従来：今後の再コールと経過未対応を2つのアイコンで分けて表示。

アップデート後：アイコンは1つになり、「経過未対応/今後の予定」の2タブでの表示。

再コールの登録方法は従来の登録方法から変更はありません。
