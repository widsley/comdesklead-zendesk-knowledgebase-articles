# 2024/07/04　IP回線の一部で発生した不具合についてのご報告

平素より大変お世話になっております。  
Widsley Supportでございます。

本日2024年07月04日にIP回線の一部のご利用ユーザー様において、IP回線の不具合が発生しておりました。

＜不具合発生日時＞  
　2024/07/04　17:14〜17:55

  
＜影響範囲／事象＞  
　一部のテナントにて、IP回線で架電ができない事象、及びIP回線での切電ができない事象が発生

  
＜原因＞  
　アプリケーションが過負荷状態となり、発信/切電ができない状態  
　  
＜対応＞  
　自然復旧（過負荷状態が軽減されたことによる）
