# 2026/03/04　Comdesk Lead夜間リリースのお知らせ

平素より大変お世話になっております。Widsley Supportでございます。

いつもComdesk Leadをご利用いただき、誠にありがとうございます。

本日（2026年3月4日）夜間帯におきまして、Comdesk Leadの機能追加および仕様変更に伴うリリースを実施いたします。

本リリースに伴い、挙動や仕様が一部変更となります。

また、各種アプリをご利用中のお客様におかれましては、最新版へのアップデートをお願いしたく存じます。

詳細につきましては、下記をご確認いただけますと幸いです。

——————————————————————————–————————————————–————————–

### **【Web / Mobile / CallServer / デスクトップアプリ 共通】**

■ログイン画面の仕様変更

パスワード入力欄に「パスワード表示ボタン（目のマーク）」を追加いたしました。

押下することで、入力中のパスワードテキストを確認できるようになります。

### **【Comdesk Lead Web】**

■オートコール機能の仕様変更

テナント設定にて、「オートコールエラー」をOFFに設定している場合

以下に該当するリストへ架電をしようとした際、自動的に次のリストへスキップされるよう仕様を変更いたしました。

・電話番号が空欄のリスト

・禁止顧客に登録されているリスト

■リスト消化率の再計算

プロジェクトへの「リスト追加時」および「リスト移動時」にも、消化率の再計算処理が実行されるようになりました。

■通話内容記録メモの文字数制限（URL欄）

架電後の通話内容記録メモにおいて、標準項目「URL」欄に100文字を超える値を入力して保存しようとした際

「URLは100文字以内で入力してください」と警告ポップアップが表示されるようになりました。

■リスト項目の作成制限

新規リスト項目を作成する際、すでに存在する項目と同一の「表示ラベル」を持つ項目は作成できない仕様に変更いたしました。

■不具合修正（カスタムユーザー種別）

システム管理者を複製して作成したカスタムユーザー種別において、一部レポートが表示されない不具合を修正いたしました。\
