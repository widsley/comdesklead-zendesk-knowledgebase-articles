# 2024/06/14　IP回線の一部で発生した不具合についてのご報告

平素より大変お世話になっております。  
Widsley Supportでございます。

本日2024年06月14日にIP回線の一部のご利用ユーザー様において、IP回線の不具合が発生しておりました。

＜不具合発生日時＞  
　2024/06/14　15:43〜16:56

  
＜影響範囲／事象＞  
　一部のテナントにて、Comdesk Phoneへの接続ができない事象、及びIP回線での架電ができない事象が発生

  
＜原因＞  
　PBX（IP回線ソフトウェア）で不正なメモリ利用が発生したため、アプリケーションが過負荷状態となり、接続ができない状態  
　  
＜対応＞  
　アプリケーション／インフラ環境を再起動し、復旧

  
ご利用中のお客様には大変ご不便、ご迷惑をお掛け致しましたこと深くお詫び申し上げます。  
なお、現在は、復旧作業が完了しており、通常通りご利用いただけます。

  
同事象の発生を防ぐべく、改善に努めさせていただきます。

その他ご不明点などございましたら、[**サポートチームまでお問い合わせ**](https://comdesklead.zendesk.com/hc/ja/requests/new)をお願いいたします。
